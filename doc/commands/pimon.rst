``pimon`` コマンド
==================

概要
----

引数
----

--help
  コマンド一覧を表示します。

-ws, --workspace
  対象となるワークスペース。

  もし指定がない場合は、次のように決定されます。

  - Linuxの場合は、 ``$HOME/.config/pimon``

COMMAND
  コマンド

コマンド一覧
------------

詳細は、 ``man pimon COMMAND`` を参照。

* ``pimon init``
* ``pimon add``
* ``pimon fetch``
* ``pimon list``
* ``pimon remove``
* ``pimon upgrade``
* ``pimon version``
